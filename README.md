# tcl
Twitter Command Line

```
Usage: tcl [options]

  Options:

    -h, --help                        output usage information
    -V, --version                     output the version number
    -o, --oauth <path to oauth.json>  A JSON file with login credentials
    -m, --method <GET|POST>           The method to use - GET or POST
    -r, --resource <resource>         The resource to use e.g. /statuses/user_timeline.json
    -p, --params <params>             OPTIONAL Twitter params in string format
```
